# Тетрис — Яндекс Игры

Классический тетрис, написанный по официальным гайдлайнам без внешних зависимостей и сборщиков. Статичные HTML / CSS / JS, готовые к публикации на платформе [Яндекс Игры](https://yandex.ru/games/).

---

## Возможности

### Механика
- **SRS** (Super Rotation System) — полная таблица wall-kicks для всех фигур, включая I-пьесу
- **7-bag randomizer** — каждые 7 фигур содержат все 7 типов ровно по одному разу
- **Hold** с блокировкой повторного использования до фиксации фигуры
- **Next ×5** — очередь из пяти следующих фигур
- **Lock delay** 500 мс, до 15 сбросов таймера на фигуру
- **T-Spin** (full / mini) — детектируется по правилу трёх углов + индексу kick'а
- **Back-to-Back ×1.5** для Tetris и T-Spin подряд
- **Combo** и **Perfect Clear** бонусы
- **Гравитация** по формуле гайдлайна: `(0.8 − (level−1) × 0.007)^(level−1)` сек/строка

### Режимы игры
- **Марафон** — бесконечная игра на рекорд с таблицей лидеров
- **Кампания** — 10 уровней с уникальными заданиями: очисти N линий, набери очки, сделай T-Spin, комбо, Back-to-Back, режим на время. До 3 звёзд за уровень.

### Интерфейс
- Четыре визуальные темы: **Неон**, **Минимал**, **Дерево**, **Ретро**
- Локализация **RU / EN** (автоопределение по SDK или `?lang=en`)
- **WebAudio-синтез** — без внешних звуковых файлов
- Адаптивная вёрстка для любого размера экрана

### Управление
| Действие | Клавиатура | Жесты (мобильный) |
|---|---|---|
| Влево / Вправо | `←` / `→` | Свайп влево / вправо |
| Поворот CW | `↑` или `X` | Тап по правой половине поля |
| Поворот CCW | `Z` или `Ctrl` | Тап по левой половине поля |
| Soft drop | `↓` | Медленный свайп вниз |
| Hard drop | `Space` | Быстрый свайп вниз или вверх |
| Hold | `Shift` или `C` | Долгое нажатие |
| Пауза | `P` или `Esc` | Два пальца |

На мобильных устройствах доступно два режима управления: **жесты** (по умолчанию) и **кнопочная панель**. Переключение — в настройках.

DAS/ARR: **133 мс / 17 мс** (стандарт гайдлайна, 8/1 кадра при 60 fps).

## Яндекс SDK

| Функция | Статус |
|---|---|
| `LoadingAPI.ready()` | После полной готовности UI |
| `GameplayAPI.start / stop` | Оборачивает каждую игровую сессию |
| Полноэкранная реклама | После Game Over, кулдаун 65 сек |
| Rewarded video | Возрождение после смерти в марафоне |
| Sticky banner | На десктопе после загрузки |
| Лидерборд `best` | Топ-10 + позиция игрока |
| `player.getData / setData` | Очки, линии, кампания, серия дней |
| `deviceInfo.type` | Определение мобильного устройства |
| Ярлык на рабочий стол | После 3-й и 10-й сыгранной игры |
| Запрос оценки | При первом новом рекорде |

Все SDK-вызовы обёрнуты в try/catch и gracefully вырождаются при отсутствии SDK (локальная разработка).

### Облачные сохранения

При загрузке данные с облака и из `localStorage` **сливаются**: числовые поля берут максимум, прогресс кампании объединяется по лучшему результату за уровень, серия дней — по более свежей дате. Это позволяет играть с нескольких устройств без потери прогресса.

Сохраняемые поля: `best`, `totalLines`, `totalGames`, `maxCombo`, `maxLevel`, `campaignJson`, `streakJson`.

## Структура проекта

```
.
├── index.html          — каркас, оверлеи, SDK-скрипт
├── css/
│   └── style.css       — все стили, один файл, четыре темы

│   ├── i18n.js         — RU/EN, data-i18n, автоопределение языка
│   ├── storage.js      — localStorage + облако, слияние при загрузке

│   ├── renderer.js     — Hi-DPI canvas, ghost, очередь, hold
│   ├── input.js        — клавиатура DAS/ARR, жесты, кнопочная панель
│   └── main.js         — бутстрап, оверлеи, кампания, реклама
└── README.md / CLAUDE.md
```

## Локальный запуск

```bash
python3 -m http.server 8080
# или
npx serve
```

Откройте `http://localhost:8080`. Добавьте `?lang=en` для английского интерфейса.

SDK при локальном запуске не загружается — реклама и лидерборд недоступны, но игра работает полностью.

## Публикация в Яндекс Игры

1. Создать игру в [консоли разработчика](https://games.yandex.ru/console).
2. Создать лидерборд с техническим именем **`best`** (зашито в `js/sdk.js` → `LB_NAME`).
3. Собрать архив (без `.git`, `CLAUDE.md`, `.claude/`):
   ```bash
   zip -r tetris.zip index.html css/ js/
   ```
4. Загрузить zip в раздел **Черновики** консоли.
5. Заполнить название, описание, загрузить иконку (512×512 PNG) и скриншот.
6. Отправить на модерацию.
